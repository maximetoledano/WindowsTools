Repères:
~~~~~~~~

Code outil:       AVR
Libellé long:     Archive Versions Repertoire
Version:          0.1
Utilisation:      Sous Windows uniquement
Points d'entrée:  Par raccourci Windows paramétré

Description:
~~~~~~~~~~~~

Procède à une sauvegarde ("photo") du répertoire passé en argument, stocké dans un fichier archive (de type zip).

Le fichier résultat est un fichier dont le nom est suffixé par ".gz", et préfixé parla date et l'heure de la photo.

Fonctionnalités disponibles:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

   * Horodatage de la photo.
   
   * Support des noms longs (quelque soit le système hôte).
   
Mode opératoire:
~~~~~~~~~~~~~~~~

   * Sélectionner le répertoire à archiver.
   
   * Lancer l'archivage du fichier (click droit sur le fichier, puis "Envoyer    vers ==> Archive Versions Repertoire").
   
Clés de registre:
~~~~~~~~~~~~~~~~~

   Aucune clé utilisée.

Evolutions:
~~~~~~~~~~~

   * 0.1: Archivage du répertoire passé en paramètre.
